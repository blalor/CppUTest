Example description
===================
This example provides a short description of how to use the TIM peripheral and
gives an example how to configure the TIM0 in PWM mode.

You can see the PWM signal generated by the timer in the P4.0 pin.


Directory contents
==================
91x_conf.h  Library Configuration file
main.c      Main program
91x_it.c    Interrupt handlers

=============
