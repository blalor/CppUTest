Example description
===================
This example provides a short description of how to use the GPIO peripheral and
gives a configuration sequence. 
All leds of GPIO3 are toggled in an infinite loop.


Directory contents
==================
91x_conf.h  Library Configuration file
91x_it.c    Interrupt handlers
main.c      Main program


Hardware environment
====================
 - Connect all pins of GPIO3  (respectively LD1..8  on STR912-SK-IAR board).
     
How to use it
=============
In order to make the program work, you must do the following :

- Run the example
