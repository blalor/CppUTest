Example description
===================

This example configures the RTC time,date and alarm.

When the alarm occurs (10 seconds from configured time), led on GPIO3.1 will start to toggle.

Directory contents
==================
91x_conf.h Library Configuration file
main.c      Main program
91x_it.c    Interrupt handlers

How to use it
=============
In order to make the program work, you must do the following :

- Run the example
